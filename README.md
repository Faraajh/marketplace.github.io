# 🛒 TirtaBekas - Marketplace Mahasiswa Untirta

Aplikasi marketplace jual beli barang bekas khusus untuk mahasiswa Universitas Sultan Ageng Tirtayasa (Untirta). Dilengkapi dengan fitur AI Image Editor untuk mempercantik foto produk.

## ✨ Fitur

- 🔍 Pencarian produk
- 📂 Filter berdasarkan kategori (Buku, Elektronik, Pakaian, Perlengkapan Kos, dll)
- 🛍️ Jual barang bekas dengan mudah
- 🖼️ AI Image Editor untuk edit foto produk
- 📱 Responsive design

## 🚀 Cara Menjalankan

**Prasyarat:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. (Opsional) Set `GEMINI_API_KEY` di file `.env.local` untuk fitur AI Image Editor

3. Jalankan aplikasi:
   ```bash
   npm run dev
   ```

4. Buka di browser: `http://localhost:3000`

## 🛠️ Tech Stack

- React 19
- TypeScript
- Vite
- TailwindCSS 4
- Lucide React Icons
- Framer Motion
